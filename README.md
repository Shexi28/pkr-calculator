# RupeeRoute — USD to PKR Withdrawal Calculator

Freelancer ka financial tool — Payoneer, nsave, ElevatePay compare karo.

## Deploy to Vercel (3 steps)

### Option A — GitHub se (Recommended)

1. GitHub par naya repo banao
2. Yeh 2 files upload karo: `index.html` aur `vercel.json`
3. [vercel.com](https://vercel.com) par jao → "New Project" → GitHub repo select karo → Deploy

### Option B — Vercel CLI se

```bash
npm install -g vercel
cd pkr-calculator
vercel
```

### Option C — Drag & Drop (Sabse aasaan)

1. [vercel.com/new](https://vercel.com/new) par jao
2. "Browse" ya drag & drop — `pkr-calculator` folder upload karo
3. Deploy button dabao — 30 second mein live!

## Features

- Custom rates — har roz khud update karo
- Jitne routes chahiye add karo
- Break-even calculator
- 6 pre-loaded examples (tumhare actual amounts)
- Best route automatically highlight hota hai

## Files

- `index.html` — poora app (single file)
- `vercel.json` — Vercel deployment config
